🎧 ฟังเสียงผู้ถามจากระบบ (BlackHole)

ให้แอปถอด เสียงผู้ถาม ที่ออกจาก Zoom/Meet (เสียงระบบ) แทนไมค์เรา — เสียงเรา (mic) จะไม่เข้า

  1. ติดตั้ง BlackHole 2ch (brew install blackhole-2ch)
  2. สร้าง Multi-Output Device (Audio MIDI Setup) = BlackHole + ลำโพง/หูฟัง → ตั้งเป็น output ของเครื่อง (จะได้ยินเอง + ส่งเข้า BlackHole)
  3. ตั้ง Input ของ Chrome = BlackHole 2ch (System Settings → Sound → Input หรือ 🔒 ข้าง URL) แล้ว รีโหลดหน้า

อยากให้เสียงเราเข้าด้วย → ใช้ Aggregate Device (BlackHole + mic จริง). ขั้นตอนเต็มดูใน README
